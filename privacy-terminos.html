SafeCell - Política de Privacidad

Titular del Tratamiento

El titular es SafeCell, contactable en: safecellitaliainfo@gmail.com

Datos recopilados

  • Datos identificativos: nombre, apellidos, código fiscal, documento de identidad;
  • Datos de contacto: email, número de teléfono;
  • Datos del dispositivo: tipo de dispositivo, número de dispositivo, IMEI;
  • Datos de autenticación: contraseña cifrada;
  • Datos de localización: coordenadas GPS en tiempo real;

Finalidades del tratamiento

Los datos se utilizan para:

  • registro y autenticación del usuario;
  • servicio de rastreo del dispositivo;
  • soporte a las Fuerzas de Seguridad previa denuncia formal;
  • gestión de comunicaciones y notificaciones de seguridad;
  • protección de la cuenta mediante autenticación biométrica;

Base jurídica

Consentimiento del usuario e interés legítimo para la protección de los dispositivos.

Conservación de datos

Los datos se eliminan definitivamente en caso de baja (disdetta).

Compartición de datos

Los datos no se venden. Pueden ser compartidos únicamente con:

  • Fuerzas de Seguridad previa verificación de la denuncia;
  • Proveedores tecnológicos (como Firebase) vinculados por obligaciones de confidencialidad.

Seguridad

Utilizamos cifrado, accesos controlados y altos estándares de protección.

Derechos del usuario (GDPR)

Acceso, rectificación, eliminación, portabilidad, oposición. Para ejercerlos: safecellitaliainfo@gmail.com

Transferencias internacionales

El servicio puede utilizar servidores de Google/Firebase fuera de la UE, protegidos mediante Cláusulas Contractuales Tipo.

Términos y Condiciones de Uso

Uso del servicio

La app ayuda al usuario a rastrear su dispositivo en caso de pérdida o robo. El rastreo activo para las Fuerzas de Seguridad se realiza solo tras verificar una denuncia. El enlace de seguimiento es válido durante 96 horas, prorrogables.

Limitaciones de responsabilidad

SafeCell no garantiza la recuperación del dispositivo perdido o robado.

Disdetta (Cancelación)

El usuario puede cancelar su cuenta y eliminar definitivamente sus datos utilizando la función “Disdetta” e introduciendo su código único.

Modificaciones

SafeCell puede modificar estos términos previa comunicación al usuario.

Ley aplicable

Reglamento GDPR y legislación italiana. Foro competente: Roma (Italia)
