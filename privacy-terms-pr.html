SafeCell – Política de Privacidad y Términos de Servicio

Política de Privacidad

Titular del Tratamiento

El titular es SafeCell, contactable en: safecellitaliainfo@gmail.com

Datos Recopilados

  • Datos identificativos: nombre, apellidos, número de seguro social/código fiscal, documento de identidad;
  • Datos de contacto: email, número de teléfono;
  • Datos del dispositivo: tipo de dispositivo, número del dispositivo, IMEI;
  • Datos de autenticación: contraseña cifrada;
  • Datos de localización: coordenadas GPS en tiempo real.

Finalidades del Tratamiento

Los datos se utilizan para:

  • registro y autenticación del usuario;
  • servicio de rastreo del dispositivo;
  • apoyo a las Fuerzas del Orden tras presentar un reporte oficial;
  • gestión de comunicaciones y notificaciones de seguridad;
  • protección de la cuenta mediante autenticación biométrica.

Base Jurídica

Consentimiento del usuario e interés legítimo para la protección de los dispositivos.

Conservación de Datos

Los datos se eliminan de forma permanente en caso de cancelación (disdetta).

Divulgación de Datos

Los datos no se venden. Solo pueden compartirse con:

  • Fuerzas del Orden, tras verificar el reporte policial;
  • proveedores tecnológicos (como Firebase) sujetos a obligaciones de confidencialidad.

Seguridad

Utilizamos cifrado, accesos controlados y altos estándares de seguridad para proteger tus datos.

Derechos del Usuario (GDPR)

El usuario tiene derecho a:

  • acceso;
  • rectificación;
  • eliminación;
  • portabilidad de datos;
  • oposición.

Para ejercer estos derechos, puedes contactar a: safecellitaliainfo@gmail.com

Transferencias Internacionales

Puede ocurrir el uso de servidores de Google/Firebase fuera de la UE bajo Cláusulas Contractuales Estándar.

Términos de Servicio

Uso del Servicio

La app ayuda al usuario a rastrear su propio dispositivo en caso de pérdida o robo. El rastreo activo para las Fuerzas del Orden se realiza solo después de verificar el reporte policial. El enlace de monitoreo es válido por 96 horas y puede extenderse.

Limitación de Responsabilidad

SafeCell no garantiza la recuperación segura o garantizada de dispositivos perdidos o robados.

Cancelación

El usuario puede revocar el servicio y eliminar todos los datos mediante la función “Disdetta” ingresando su código único.

Modificaciones

SafeCell puede modificar este aviso, proporcionando comunicación previa cuando sea necesario.

Ley Aplicable

Ley aplicable: GDPR y ley italiana – tribunal competente: Roma (Italia).

Contacto

Para cualquier consulta, puedes contactar: safecellitaliainfo@gmail.com
